《雄安时空变迁与建设情报平台》项目可行性研究报告
1. 引言与背景分析
1.1 项目背景
自2017年雄安新区设立以来，大规模的城市基础设施建设与大规模的原住民征迁同步推进。截至2026年，新区的核心起步区、容东片区、昝岗片区等已由规划蓝图全面走向大规模实景交付。在这个过程中，沉淀了两个核心群体的强烈痛点：一是数以十万计的回迁居民、本地原住民对“消失的历史家园”有着强烈的精神寻根与时空变迁记忆留存需求；二是面向雄安新区的数十万基建供应链企业（建材商、五金店、重型机械租赁商、劳务公司等），在面对工地高频流转、封闭式施工以及动态项目部搬迁时，存在极其严重的基建商机“信息差”。
本项目的技术底座源自技术成果 
20170813036_王自立_基于WebGIS的雄安新区建设成果展示系统的设计与实现.doc
。该成果成功跑通了基于开源 WebGIS 的空间数据可视化、图层动态切换与属性查询的核心闭环。本项目旨在保留该系统的空间地理信息处理内核的基础上，引入现代大语言模型（LLM）与 
AI Agent（智能体）集群
 技术，进行面向双边市场（C端民生情怀 + B端基建商机）的跨代升级与商业化落地。

2. 市场可行性分析
平台采用双边市场（Two-Sided Market）模型，通过C端情怀功能聚集低成本的高频流量，再通过B端供应链刚需情报卡死高净值变现通道。
2.1 C端市场：征迁记忆与回迁民生服务
目标用户
：雄安新区容东、容西、雄东片区数十万回迁原住民，以及密切关注雄安变迁的网民。
用户痛点
：由于老村庄全面拆迁，空间地貌彻底改变，原住民在回迁高楼后存在极强的历史寻根、老照片留存、村史怀旧的情感刚需。同时，大批原住民面临新居安置、房屋租赁、本地生活采购等高频现实需求。
市场空间
：本地熟人社交与同城情怀流量极易通过微信小程序进行裂变，获客成本接近为零。
2.2 B端市场：基建供应链与腹地产业联动（核心变现池）
目标用户
：进驻雄安新区的央企/国企总包项目部，以及雄安本地及周边大后方（如
保定、定州
等区域）的建材、劳务和机械设备供应商。
区域定位
产业供应链角色
市场痛点
雄安核心区
总包项目部、直接基建现场
施工现场全封闭，临时项目部频繁搬迁，就近采购响应慢，物流车辆限行路线不透明。
保定/定州等腹地
重型机械租赁商（挖机、塔吊）、大宗建材厂、工程网片与劳务输出基地
处于雄安建设大后方，信息不对称。渴望将物资与设备打入雄安市场，但无法实时掌握工地的精准点位和进场时机。
商机价值
：B端企业的核心痛点是“抢进场时机”。一个重型机械租赁商如果能提前1-3个月掌握某个地块的征迁平整进度，其成单概率将提升50%以上。这种高价值的商业情报使B端企业具备极高的付费意愿。

4. 法律与合规红线规避方案（重中之重）
在雄安新区从事地理信息与建设情报开发，合规是项目的生命线。本项目建立了严格的红线防火墙：
4.1 规避非法测绘红线
合规解法
：平台
绝不进行任何自主测绘与矢量底图编制
。平台底层完全通过 OpenLayers 接口，合规引用国家法定的
天地图
或腾讯、高德的公开开放底图。平台仅在法定底图之上，以数据图层（Marker 点、GeoJSON 覆盖物）的形式叠加业务属性数据，完全符合国家测绘地理信息法规。
4.2 规避国家涉密与保密工程红线
合规解法
：AI Agent 的数据源采取“100%公开白名单”机制。智能体仅被允许抓取政府公共资源交易网、官方招投标平台、央企官网公开新闻等互联网已公开的文字信息。
脱敏处理
：平台禁止收录任何未经官方发布的内部规划图、地下管廊涉密坐标或军事防汛敏感区域。地图打点均采用民用级脱敏坐标，将风险降为零。
5. 经济效益与财务可行性分析
5.1 财务与投资回报率（ROI）数学模型
为了精算独立开发者的运营成本与盈利前景，建立以下财务模型。
5.1.1 运营总成本（Monthly Cost）模型
设月度运营总成本为 Ctotal​，由云服务器固定成本 Sserver​ 与 AI Agent 大模型 API 消耗成本 Allm​ 组成：
Ctotal​=Sserver​+Allm​
Sserver​：包含 4核8G 云服务器（运行 FastAPI、GeoServer、PostGIS）及对象存储，约为 300 元/月。
Allm​：设 Agent 每日抓取清洗 100 条公告，每条公告消耗 5,000 tokens（含 Prompt 与上下文），单价以国内主流大模型 2026 年平均价格 0.01元/万 tokens 计算。每日成本约 0.5 元，月度 Allm​≈15 元。
月度固定运营成本 Ctotal​≈315 元
，财务压力极低。
5.1.2 预期收益（Monthly Revenue）模型
设月度总收益为 Rtotal​，由B端高亮入驻费、VIP情报会员费及C端流量主广告费组成：
Rtotal​=(Nshop​×Pshop​)+(Nvip​×Pvip​)+Rads​
Nshop​：雄安及周边保定、定州入驻的五金/机械/劳务店铺数。
Pshop​：入驻费（单店定价 300 元/年，折合 25 元/月）。
Nvip​：订阅高级建设情报的 VIP 供应链会员数。
Pvip​：会员费（定价 1,188 元/年，折合 99 元/月）。
Rads​：微信小程序流量主月度广告收益（预估 500 元/月）。
5.1.3 收支平衡点与投资回报率（ROI）
若平台在中期（第 6 个月）达到以下温和指标：保定/定州及本地入驻商户 Nshop​=80 家，VIP 订阅会员 Nvip​=30 家。 则月度总收益为：
Rtotal​=(80×25)+(30×99)+500=2000+2970+500=5470 元/月
年度投资回报率（ROI）测算公式为：
ROI=Cinitial​∑t=112​(Rtotal,t​−Ctotal,t​)​×100%
设前期初始开发投入（个人时间折算成本及软件注册费等）Cinitial​=5000 元，则首年即可实现：
ROI=5000(5470−315)×12​×100%=500061860​×100%=1237.2%
模型数据表明，该项目具有极高的高毛利与现金流回报特性。
